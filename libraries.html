Reference Directory

대표적인 영상처리 라이브러리와 제품군

아래 항목은 참고용 소개입니다. 특정 제품 판매 페이지가 아니라 문제 유형과 운영 조건에 따라 검토할 수 있는 대표 선택지를 정리한 것입니다.

Open Source

OpenCV

범용 오픈소스 컴퓨터비전 라이브러리입니다. 빠른 프로토타이핑과 커스텀 알고리즘 구현에 적합합니다.

공식 사이트
Commercial

HALCON

산업용 머신비전에서 널리 쓰이는 상용 라이브러리로 다양한 검사, 측정, 정렬 기능을 제공합니다.

공식 사이트
No-Code

MERLIC

코드 작성 부담을 낮춘 GUI 기반 비전 소프트웨어로 빠른 현장 구성과 운영에 유리합니다.

공식 사이트
Commercial

Cognex VisionPro

PC 기반 머신비전 애플리케이션 구축에 많이 쓰이는 상용 툴킷입니다.

공식 사이트
SDK

Basler pylon

카메라 제어와 이미지 획득을 위한 SDK 및 운영 툴 모음으로 Basler 카메라 환경에서 자주 사용됩니다.

공식 사이트
SDK

IDS peak

IDS 카메라를 위한 SDK 및 운영 툴로 GigE Vision, USB3 Vision 통합에 적합합니다.

공식 사이트
Inference Stack

CUDA / TensorRT

딥러닝 기반 추론 속도 최적화가 필요한 환경에서 자주 검토되는 GPU 가속 스택입니다.

공식 사이트
Camera SDK

Hikrobot MVS

Hikrobot 카메라 제어 및 이미지 획득용 SDK 계열로 현장 장비 구성에 따라 검토됩니다.

공식 사이트
상표 고지
