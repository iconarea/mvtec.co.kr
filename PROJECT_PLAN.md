# MVTEC.CO.KR 홈페이지 기획서

## 1. 프로젝트 목적

`mvtec.co.kr`는 머신비전 컨설팅, 영상처리 기술 소개, 적용 사례 제안, 관련 라이브러리 및 제품 안내를 중심으로 한 전문 소개 사이트로 기획한다.

이 사이트의 핵심 목적은 다음과 같다.

- 머신비전 도입이 필요한 기업에 기술 컨설팅 이미지를 제공
- 검사 자동화, 측정, 인식, 추적, 분류 등 영상처리 문제 해결 역량을 소개
- 주요 영상처리 라이브러리와 상용 제품을 비교적 중립적으로 안내
- 상담 문의와 프로젝트 의뢰를 유도

## 2. 가장 중요한 법적 주의사항

이 프로젝트는 이름상 `MVTec`과 혼동될 가능성이 있으므로, 기획 단계부터 아래를 강하게 반영해야 한다.

- 본 사이트는 유럽의 `MVTec Software GmbH`와 무관함을 명시
- `HALCON`, `MERLIC`, `MVTec` 등 타사의 상표를 자사 브랜드처럼 보이게 사용하지 않음
- 소개 대상 제품은 `참고용 제품 소개` 또는 `연결 링크 안내`로만 표기
- 사이트 상단 또는 하단에 고정 문구로 `본 사이트는 독립적인 머신비전 컨설팅 사이트이며, 독일 MVTec Software GmbH와 관계가 없습니다`를 명시
- 도메인/브랜드 사용이 상표 오인 가능성을 높인다면, 실제 대외 운영 전 상표·표시 리스크 검토 필요
- 로고, 제품 화면, 자료 이미지 사용 시 저작권 및 상표권 확인 필요

권장 문구 예시:

```text
본 사이트는 독립적인 머신비전 컨설팅 및 정보 제공 사이트이며,
유럽의 MVTec Software GmbH 및 그 제품과는 별개의 주체입니다.
각 제품명과 상표는 해당 권리자에게 귀속됩니다.
```

## 3. 사이트 포지셔닝

추천 포지셔닝은 `머신비전 기술 컨설팅 + 솔루션 기획 + 라이브러리 정보 허브`이다.

너무 판매 위주로 보이기보다는 아래 톤이 적합하다.

- 기술적이고 차분한 인상
- 산업 자동화와 검사 시스템 경험을 강조하는 문구
- 특정 제품 판매상보다 `문제 해결형 컨설팅` 이미지 우선
- 방문자가 `무엇을 할 수 있는 회사인지`를 10초 안에 이해 가능해야 함

## 4. 메인 메시지 방향

메인 첫 화면에서 전달할 메시지는 다음 구조가 좋다.

- 머신비전 컨설팅과 영상처리 적용 설계
- 검사 자동화, 측정, OCR, 바코드, 결함 검출, 위치 보정 등 핵심 분야 제시
- 다양한 영상처리 라이브러리와 제품을 비교 검토해 최적 구성을 제안한다는 메시지
- 상담 문의 CTA 배치

메인 카피 예시:

```text
머신비전 문제를 기술적으로 분석하고,
적합한 영상처리 라이브러리와 시스템 구성을 제안합니다.
```
